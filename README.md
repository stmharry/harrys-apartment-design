# Harry's Apartment Design

Static public deployment package for a personal interior style panorama gallery.

## Cloudflare Pages

- Framework preset: None
- Production branch: main
- Build command: leave empty or use `exit 0`
- Build output directory: `public`

The site is static HTML/CSS/JS plus local JPEG and JSON assets. Pannellum is loaded from a pinned jsDelivr CDN URL.
